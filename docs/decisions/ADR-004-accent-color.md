# ADR-004: Accent cam hổ phách #F5A623, không dùng đỏ Netflix
- **Ngày:** 2026-07-02 · **Trạng thái:** Accepted
## Bối cảnh
UI "web xem phim" nền tối dễ trượt thành clone Netflix (đỏ #E50914) — vừa nhạt nhòa vừa rủi ro nhận diện thương hiệu.
## Quyết định
Accent chính #F5A623 (cam hổ phách), badge 4K vàng đồng #C9A227 — bảng màu riêng của VietArr.
## Hệ quả / đánh đổi
Nhận diện riêng, đỏ được giải phóng để dành cho trạng thái lỗi đúng ngữ nghĩa.
