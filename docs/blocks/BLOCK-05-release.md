# BLOCK-05 — ĐÓNG GÓI PHÁT HÀNH v1.0
> **Trạng thái:** PLANNED · **Phụ thuộc:** B1–B4

## 1. Vision
Người lạ (biết dùng terminal cơ bản) cài được VietArr trong 15 phút không cần hỏi ai. Repo public chuyên nghiệp.

## 2. Scope & Non-Goals
**Scope:** one-liner `curl | bash` (có checksum) · docs người dùng tiếng Việt + video ngắn · README song ngữ · CONTRIBUTING.md, LICENSE, issue templates · GitHub Actions: lint + build + release tag · security review toàn bộ (checklist SECURITY.md) · test cài trên máy ngoài lab.
**Non-Goals:** KHÔNG marketing/website riêng · KHÔNG hỗ trợ OS mới.

## 5–9.
Điền khi IN PROGRESS. **DoD khung:** 2 người ngoài cài thành công độc lập · `npm audit` sạch mức high · không secret nào trong lịch sử git (chạy gitleaks).
